# TVEM-intrinsic-capacity

Stata, Mplus, R, and SAS syntax for "Dynamic associations between intrinsic capacity and disability before death among older adults"

The repository contains the Stata, Mplus, R, and SAS syntax to replicate all of the analyses from our publication

There are five files:

2025-10-10_Mplus-input_v01.docx # This is the annotated Mplus input file in Microsoft Word document format
2025-10-10_R-script_v01.R # This is the annotated R script
2025-10-10_SAS-input_v01.docx # This is the annotated SAS input file in Microsoft Word document format
2025-10-10_Stata-do-file_v01.do # This is the annotated Stata do file
2025-10-10_Stata-do-file_v01.pdf # This is the annotated Stata do file in PDF format
